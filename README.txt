Rendu
-------------------------------------------------
Toutes les fonctionnalités ont été implémentées
jusqu'au 'for' compris.


Nous avons eu des difficultés à implémenter la synthaxe demandée
du 'if then else'. Ainsi, pour des soucis d'ambiguïté nous avons
dû rajouter des parenthèses.
La synthaxe est donc la suivante :

if (expression) then (expression) else (expression)


Sinon pour tout le reste la synthaxe est celle demandée, ou en
tout cas, une qui correspond aux exemples donnés.

Il serait par ailleurs agréable d'avoir plusieurs fichiers de
test plus fournis que les quelques exemples disponibles sur le git.
Il est en effet vraiment compliqué de définir une synthaxe précise
à partir de 3 lignes.

Exemple
-------
Les seuls exemples pour le 'for' à notre disposition étant :
- for i := 2 to 3 do print(i)
- for i := 2 to 1 do print(i)

Nous en déduisons donc que les 'for' n'ont qu'une seule instruction.


Notre projet n'a malheureusement aucune extravagance.

-------------------------------------------------
Au final
-------------------------------------------------
Malgré le temps que nécessite ce projet nous avons su avancer et garder
la tête froide pour ne pas succomber à l'envie de laisser tomber.
Ainsi, nous avons vaillamment fini dans les temps !
